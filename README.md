> A React demo project with Server Side Rendering.

## Usage
1. Install packages.
```
npm install
```

2. Run the project.
```
npm run dev
```

or

```
# Build the server
npm run dev:build:server

# Build the client (with event listener)
npm run dev:build:client

# Start Inspect for build/bundle.js
npm run dev:server
```
